# BookingYa_tdd
Proyecto BookingYa TDD - gestión de reservas (Fase 1)

# DESCRIPCIÓN
Este proyecto tiene como objetivo aplicar el enfoque de desarrollo guiado por pruebas (TDD) para garantizar la calidad del software en la gestión de reservas dentro de una plataforma digital.

## Entre sus principales funcionalidades se incluyen:

-Creación de una reserva

-Actualización de una reserva existente

-Eliminación (cancelación) de una reserva

-Consulta de todas las reservas

-Obtención de una reserva por ID

-Consulta de reservas por usuario
